# ⚔️ Dark Fantasy Breakout

A Breakout game where you need to break all the tiles in order to advance and win. Ambiented in a Dark Fantasy World where you move a magic sword to destroy this magic blocks where you sometimes meet blessings or curses 

## 🕹️ How To Play

The Objective is to destroy every block of every level without losing your lives. There are **Special Blocks** that will alter the gameplay.

### Controls
* **A**: Move the sword to the **left**
* **D**: Move the sword to the **right**
* **Spacebar**: Start the game
* **F5**: Restart the game

### Special Mechanics
* **Freezing ball Blessing**: When a golden block is hit, the ball will appear frozen, slowing it down by half its speed, after 5 seconds the ball will defrost with 1.75 times its frozen speed.
* **Extra Lives**: You start with 3 lives, but when a green block is hit, an extra live will be granted
* **Freezing Sword Curse**: When a frozen block is hit, your sword will appear frozen and lose half its speed, after 5 seconds this curse is reverted.
* **Progression**: The ball speed will increase slightly with each bounce.

## 🚀 How to run the Game

### Requirements
* Make Sure the game files are organized this way:
    ```text
    /Game Folder
    ├── /Assets            # Images (.png) & Sounds (.wav, .mp3)
    ├──/libs               #Functions and Object libraries
    ├── breakout.html         # File to run the game
    ├── breakoutjs.js      # Game logics
    └── styles.css      # Game style
    ```

### Running the game
1. **Clone** every file inside the same folder.
2. Make sure you aren't missing assets.
3. Double click the file `breakout.html`
4. The Game wil open automatically on your browser
